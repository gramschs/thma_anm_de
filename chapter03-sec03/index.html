3.3 Anwendung: Wärmeübertragung in einer Mehrschichtwand

Bisher haben wir LGS an einem Obstmarkt-Beispiel geübt. Jetzt wenden wir dasselbe Werkzeug auf ein Problem aus dem Maschinenbau an: eine Außenwand aus drei Schichten mit unterschiedlichen Wärmedurchgangswiderständen. Wie hoch ist die Temperatur an den Grenzflächen, und wie groß ist der Wärmestrom?

Wir werden sehen, dass der Weg von den physikalischen Gleichungen zur Matrix immer gleich ist: Unbekannte auf die linke Seite, bekannte Größen auf die rechte.

Lernziele¶

Lernziele

  • Sie können physikalische Gleichgewichtsbedingungen in ein LGS A⋅x⃗=b⃗\mathbf{A} \cdot \vec{x} = \vec{b}A⋅x=b überführen.

  • Sie können die Matrix A\mathbf{A}A und den Vektor b⃗\vec{b}b aus umgeformten Gleichungen ablesen.

  • Sie können das LGS lösen und das Ergebnis physikalisch interpretieren.

Das physikalische Modell¶

Wir betrachten eine Wand aus drei Schichten A, B, C mit den thermischen Widerständen RAR_ARA​, RBR_BRB​, RCR_CRC​ in K/W. Links herrscht die Temperatur TLAT_{LA}TLA​, rechts TCRT_{CR}TCR​.

Im stationären Zustand ist der Wärmestrom QQQ durch alle Schichten gleich. Das Wärmeübertragungsgesetz (analog zum Ohmschen Gesetz) lautet für jede Schicht:

Q=ΔTiRiQ = \frac{\Delta T_i}{R_i}Q=Ri​ΔTi​​

Das liefert drei Gleichungen mit drei Unbekannten (TABT_{AB}TAB​, TBCT_{BC}TBC​, QQQ):

TLA−TABRA=Q(1)\frac{T_{LA} - T_{AB}}{R_A} = Q \qquad (1)RA​TLA​−TAB​​=Q(1)
TAB−TBCRB=Q(2)\frac{T_{AB} - T_{BC}}{R_B} = Q \qquad (2)RB​TAB​−TBC​​=Q(2)
TBC−TCRRC=Q(3)\frac{T_{BC} - T_{CR}}{R_C} = Q \qquad (3)RC​TBC​−TCR​​=Q(3)

Von den Gleichungen zur Matrixform¶

Wir bringen alle Unbekannten auf die linke Seite. Dafür multiplizieren wir jede Gleichung mit RiR_iRi​ und ordnen um:

TAB+RA⋅Q=TLA(1′)T_{AB} + R_A \cdot Q = T_{LA} \qquad (1')TAB​+RA​⋅Q=TLA​(1′)
−TAB+TBC+RB⋅Q=0(2′)-T_{AB} + T_{BC} + R_B \cdot Q = 0 \qquad (2')−TAB​+TBC​+RB​⋅Q=0(2′)
−TBC+RC⋅Q=−TCR(3′)-T_{BC} + R_C \cdot Q = -T_{CR} \qquad (3')−TBC​+RC​⋅Q=−TCR​(3′)

Jetzt lesen wir die Koeffizientenmatrix zeilenweise ab. Der Lösungsvektor ist x⃗=(TAB, TBC, Q)T\vec{x} = (T_{AB},\, T_{BC},\, Q)^Tx=(TAB​,TBC​,Q)T:

(+10RA−1+1RB0−1RC)⏟A⋅(TABTBCQ)⏟x⃗=(TLA0−TCR)⏟b⃗\underbrace{\begin{pmatrix} +1 & 0 & R_A \\ -1 & +1 & R_B \\ 0 & -1 & R_C \end{pmatrix}}_{\mathbf{A}} \cdot \underbrace{\begin{pmatrix} T_{AB} \\ T_{BC} \\ Q \end{pmatrix}}_{\vec{x}} = \underbrace{\begin{pmatrix} T_{LA} \\ 0 \\ -T_{CR} \end{pmatrix}}_{\vec{b}}A⎝⎛​+1−10​0+1−1​RA​RB​RC​​⎠⎞​​​⋅x⎝⎛​TAB​TBC​Q​⎠⎞​​​=b⎝⎛​TLA​0−TCR​​⎠⎞​​​
Rezept: Matrixform ablesen

Jede umgeformte Gleichung liefert eine Zeile von A\mathbf{A}A und einen Eintrag in b⃗\vec{b}b. Der Koeffizient der jjj-ten Unbekannten in der iii-ten Gleichung steht in AijA_{ij}Aij​. Unbekannte, die nicht vorkommen, erhalten den Koeffizienten 0.

Implementierung und Lösung¶

import numpy as np

# ---------- gegebene Größen ----------
R_A  = 0.5   # thermischer Widerstand Schicht A in K/W
R_B  = 0.3   # thermischer Widerstand Schicht B in K/W
R_C  = 0.7   # thermischer Widerstand Schicht C in K/W
T_LA = 300.  # Temperatur linke Seite in K
T_CR = 310.  # Temperatur rechte Seite in K

# ---------- Koeffizientenmatrix aufstellen ----------
# Unbekannte: x = [T_AB, T_BC, Q]
#
# Gleichung (1') aus  (T_LA - T_AB)/R_A = Q
#   => T_AB + 0*T_BC + R_A*Q = T_LA
#   => Zeile 1: [+1,  0, R_A]  |  b[0] = T_LA
#
# Gleichung (2') aus  (T_AB - T_BC)/R_B = Q
#   => -T_AB + T_BC + R_B*Q = 0
#   => Zeile 2: [-1, +1, R_B]  |  b[1] = 0
#
# Gleichung (3') aus  (T_BC - T_CR)/R_C = Q
#   => 0*T_AB - T_BC + R_C*Q = -T_CR
#   => Zeile 3: [ 0, -1, R_C]  |  b[2] = -T_CR
A = np.array([
    [+1.,  0., R_A],
    [-1., +1., R_B],
    [ 0., -1., R_C],
])
b = np.array([T_LA, 0., -T_CR])

# ---------- Lösbarkeit prüfen (wie in Kapitel 3.1) ----------
det_A = np.linalg.det(A)
print(f'Determinante: {det_A:.4f}')
if np.isclose(det_A, 0.):
    print('Keine eindeutige Lösung.')
else:
    print('Eindeutige Lösung vorhanden.')

# ---------- Lösen und Ergebnis ausgeben (wie in Kapitel 3.2) ----------
x = np.linalg.solve(A, b)
T_AB, T_BC, Q = x   # Ergebnis in drei Variablen entpacken

print(f'\nT_AB = {T_AB:.2f} K   (Grenzfläche A-B)')
print(f'T_BC = {T_BC:.2f} K   (Grenzfläche B-C)')
print(f'Q    = {Q:.4f} W   (Wärmestrom)')

# ---------- Probe ----------
print('\nProbe bestanden:', np.allclose(A @ x, b))

Das negative Vorzeichen von QQQ ist kein Fehler: in unserem Ansatz zeigt die positive Richtung von links nach rechts, aber die Wärme fließt von der wärmeren rechten Seite (TCR=310T_{CR} = 310TCR​=310 K) zur kälteren linken Seite (TLA=300T_{LA} = 300TLA​=300 K). Die Abbildung zeigt diesen physikalischen Fluss von rechts nach links. Beides ist konsistent.

Zur Kontrolle berechnen wir noch die Temperaturdifferenz über jede Schicht. Schicht C hat den größten Widerstand und sollte daher den größten Temperatursprung liefern, genauso wie der größte Widerstand in einem elektrischen Schaltkreis den größten Spannungsabfall erzeugt.

delta_B = T_BC - T_AB
delta_C = T_CR - T_BC

Lösung

Zu Frage 1: Der Eintrag A21=−1A_{21} = -1A21​=−1 stammt aus Gleichung (2’), die aus (TAB−TBC)/RB=Q(T_{AB} - T_{BC})/R_B = Q(TAB​−TBC​)/RB​=Q durch Umformen entsteht. Nach links gebracht steht −TAB-T_{AB}−TAB​ am Anfang. Der Koeffizient ist negativ, weil TABT_{AB}TAB​ in Gleichung (2) mit negativem Vorzeichen auf der linken Seite erscheint: die Temperaturdifferenz in Schicht B ist TAB−TBCT_{AB} - T_{BC}TAB​−TBC​, also nimmt TABT_{AB}TAB​ dort den Platz des Minuenden ein, der nach dem Umformen auf die linke Seite mit umgekehrtem Vorzeichen landet.

Zu Frage 2: b[1]=0b[1] = 0b[1]=0, weil Gleichung (2’) keine bekannte Größe auf der rechten Seite enthält. Die mittlere Schicht ist nur von Grenzflächen umgeben, deren Temperaturen selbst unbekannt sind. Es gibt keine fest vorgegebene Randbedingung, die direkt in diese Gleichung eingeht.

Zu Fragen 3 und 4:

import numpy as np

# Gegebene Größen
R_A  = 0.2
R_B  = 1.8
R_C  = 0.05
T_LA = 268.   # Innen (kalt)
T_CR = 293.   # Aussen (warm)

# Koeffizientenmatrix: gleiche Struktur wie im Beispiel oben
A = np.array([
    [+1.,  0., R_A],
    [-1., +1., R_B],
    [ 0., -1., R_C],
])
b = np.array([T_LA, 0., -T_CR])

# Lösung
x = np.linalg.solve(A, b)
T_AB, T_BC, Q = x

print(f'T_AB = {T_AB:.2f} K  ({T_AB - 273.15:.1f} °C)')
print(f'T_BC = {T_BC:.2f} K  ({T_BC - 273.15:.1f} °C)')
print(f'Q    = {Q:.4f} W')
print('Probe bestanden:', np.allclose(A @ x, b))

print(f'\nTemperaturabfall Schicht A (Beton):   {T_AB - T_LA:.2f} K')
print(f'Temperaturabfall Schicht B (PU-Foam): {T_BC - T_AB:.2f} K')
print(f'Temperaturabfall Schicht C (Stahl):   {T_CR - T_BC:.2f} K')

Den größten Temperaturabfall hat Schicht B (Polyurethan-Schaum), weil ihr thermischer Widerstand mit RB=1,8R_B = 1{,}8RB​=1,8 K/W bei weitem am größten ist. Das ist genau der Sinn einer Wärmedämmung: sie übernimmt den Großteil der Temperaturdifferenz zwischen innen und außen.

Zusammenfassung und Ausblick¶

Das Vorgehen ist immer dasselbe: physikalische Gleichungen umformen, alle Unbekannten nach links, alle bekannten Größen nach rechts, dann A\mathbf{A}A und b⃗\vec{b}b ablesen. np.linalg.solve liefert die Lösung, np.allclose sichert sie ab.

Dieses Muster lässt sich auf beliebig komplexe Systeme übertragen. Im nächsten Kapitel sammeln wir Übungsaufgaben zu LGS, bevor wir es in Kapitel 3.5 auf elektrische Netzwerke anwenden.
