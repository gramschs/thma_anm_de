## Section 2.2 — Introduction to Matplotlib

**Matplotlib** is the standard plotting library for Python.  
Its `pyplot` sub-module (conventionally imported as `plt`) provides a MATLAB-style interface
for creating publication-quality figures with minimal code.

## Learning Objectives
After completing this notebook you will be able to:
- Create line plots and scatter plots using `ax.plot()` and `ax.scatter()`.
- Customise figures: titles, axis labels, limits, legends, and grids.
- Arrange multiple subplots with `plt.subplots(nrows, ncols)`.
- Add error bars with `ax.errorbar()`.
- Produce simple animations using `FuncAnimation`.

```python
import numpy as np
import matplotlib.pyplot as plt
```

## 2.2.1 Creating a Simple Line Plot

The standard Matplotlib workflow is:

1. Create a **figure** and one or more **axes** with `fig, ax = plt.subplots()`.
2. Call plotting methods on the axis object (`ax.plot(...)`, `ax.scatter(...)`, …).
3. Decorate the plot (labels, title, legend, grid).
4. Display with `plt.show()` and/or save with `fig.savefig(...)`.

## 2.2.2 Multiple Subplots

`plt.subplots(nrows, ncols)` returns a **figure** and an **array of axes**.  
Access individual axes with `ax[i]` (1-D layout) or `ax[i, j]` (2-D grid).

```python
fig, ax = plt.subplots(nrows=1, ncols=2, figsize=(10, 4))
ax[0].plot(...)   # left subplot
ax[1].plot(...)   # right subplot
```

`plt.tight_layout()` adjusts spacing between subplots automatically.

## 2.2.5 Animations with FuncAnimation

Matplotlib can produce animations using `FuncAnimation` from `matplotlib.animation`.

**Basic idea:**
1. Create the figure and initial plot objects.
2. Write an `update(frame)` function that modifies those objects for each frame.
3. Pass figure, update function, and frame list to `FuncAnimation`.
4. Save as GIF or MP4.

Study the sine-wave example below:

```python
from matplotlib.animation import FuncAnimation

# Wave parameters
VELOCITY   = 1.0   # m/s
WAVENUMBER = 1.0   # 1/m
DT         = 0.1   # seconds per frame

x_grid = np.linspace(start=0, stop=2 * np.pi, num=100)

fig, ax = plt.subplots(figsize=(8, 3))
line, = ax.plot(x_grid, np.sin(x_grid), color='black')
ax.set_xlim(0, 2 * np.pi)
ax.set_ylim(-1.2, 1.2)
ax.set_xlabel('x in m')
ax.set_ylabel('sin(x)')
ax.set_title('Travelling sine wave')
ax.grid(True)

def update_sine(frame):
    time_elapsed = frame * DT
    y_vals = np.sin(WAVENUMBER * (x_grid + VELOCITY * time_elapsed))
    line.set_data(x_grid, y_vals)
    return line,

ani = FuncAnimation(fig, update_sine, frames=range(1, 100), blit=True)
ani.save('sine_wave_animation.gif', writer='pillow', fps=int(1.0 / DT))
plt.show()
```
